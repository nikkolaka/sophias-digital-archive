About Me

About the Author

Hi, my name is Sophia. I am a high school student, c/o 23. My goal with this website is to learn html, css, javascript, and other things along the way.


Website History

This is the second iteration of this website. The first prototype was ugly and poorly structured so I restarted with what I learned.

As you can tell, this website is inspired by the early internet simplle web design. I used early 2000s web templates as reference but using newer web dev techniques. I like the style because of how simple it is to learn.

Honestly, I was inspired to make this website after finding Paul Nylander's bugman123.com website. I found his website after searching for cool pictures of fractals and hyperbolic art.

Interests

As you may have guessed I enjoy creating this website. My brother is a computer science major which lead me to find interest in coding and stuff. I also took a CS class in highschool which further grew my interest. To be honest, I like the design aspect of web development or programming.
